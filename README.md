# CopyText ✨

## 🌐 Live Demo

👉 

---

Modern dark UI **Copy to Clipboard tool** built with pure HTML, CSS, and JavaScript.
Focused on clean design, smooth interactions, and real-world usability.

---

## 🚀 Features

* 📋 Copy text instantly to clipboard
* ⌨️ Keyboard shortcut: **Ctrl + Enter**
* 🔔 Toast notification with animated progress bar
* 🔢 Live character counter
* 🚫 Disabled copy button when input is empty
* 🧹 Clear text functionality
* 🎯 Auto focus on load

---

## 🎮 How to Use

1. Type or paste text into the input field
2. Click **Copy Text** or press **Ctrl + Enter**
3. Text is copied instantly
4. Use **Clear** to reset input
